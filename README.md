# Крестики-Нолики (Tower-Tac-Toe)

Стратегическая вариация классических крестиков-ноликов с механикой башен, лимитом фишек и динамическим полем.

## 🎮 Правила игры

### 1. Лимит фишек
У каждого игрока на поле может быть одновременно не более **3 фишек**. 
Когда вы ставите свою **4-ю фишку**, ваша самая первая (старая) фишка автоматически исчезает («испаряется»).

### 2. Индикатор жизни
В правом нижнем углу каждой фишки отображается число — количество ходов до её исчезновения:
- 🟢 **3**: Фишка только что поставлена.
- 🟡 **2**: Осталось 2 хода.
- 🔴 **1**: Фишка исчезнет при вашем следующем ходе.

### 3. Обвал башни
Ячейка полностью очищается (все фишки в ней исчезают), если:
- В ячейке накапливается **4 фишки**.
- Фишка с индикатором **1** находится на вершине башни высотой 3 в момент своего исчезновения.

### 4. Условие победы
Побеждает тот, кто первым выстроит ряд из 3 своих символов (по горизонтали, вертикали или диагонали). 
**Важно:** учитываются только **верхние** фишки в каждой ячейке.

## 🚀 Технологии
- **React 18** + **TypeScript**
- **Tailwind CSS** (стилизация)
- **Framer Motion** (анимации)
- **Lucide React** (иконки)

## 🛠 Установка и запуск
1. Установите зависимости: `npm install`
2. Запустите сервер разработки: `npm run dev`
3. Откройте `http://localhost:3000` в браузере.
